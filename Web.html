Web 2.0 und Web 3.0

Das Internet hat sich über die Jahre stark verändert und wurde von Web 1.0 zu Web 2.0 entwickelt und wird in Zukunft wahrscheinlich zu Web 3.0 entwickelt werden.

Von Web 1.0 zu Web 2.0 hat sich für die private Person hat sich hauptsächlich geändert, dass jeder Informationen zu Websites hinzufügen kann (z.B. Soziale Medien).

Web 2.0

Der Begriff Web 2.0 wurde erstmals 2003 von Eric Knorr erwähnt. Es ist die Entwicklung aus Web 1.0 und bedeutet, dass jeder eine Seiten mitgestalten kann. Ein modernerer und weiter verbreiteter begriff für Web 2.0 ist social Web beziehungsweise social Media. Bei Web 2.0 steht die Kommunikation im Mittelpunkt im Gegensatz zu den statischen Seiten des Web 1.0.

Web 3.0

Die erste Erwähnung von Web 3.0 erfolgte im Jahr 2014 durch den Mitgründer von Ethereum, Gavin Wood. Es handelt sich um eine visionäre Vorstellung des Internets, bei der das Netzwerk dezentralisiert ist und auf vielen einzelnen Endgeräten betrieben wird. Im Gegensatz zum zentralisierten Internet können bei Web 3.0 keine Daten gestohlen werden, da kein großes Unternehmen den Server besitzt. Stattdessen hat jeder Benutzer einen Teil des Netzwerks auf seinem eigenen PC. Anstelle von normalen Währungen sollen Kryptowährungen verwendet werden, da auch sie dezentral sind und nicht nachverfolgbar. Es gibt jedoch auch einige Nachteile, die Experten davon abhalten, sicher zu sein, dass sich das Internet zu Web 3.0 Entwickeln wird. Der größte Punkt, der dagegenspricht, ist, dass das jetzige Internet sehr anders aufgebaut ist und ein solcher Wechsel daher sehr schwierig wäre. Außerdem haben Blockchains wie Kryptowährungen einen hohen Energiebedarf, da sie eine sehr aufwändige Verschlüsselung haben.
